## ⚖️ OWNERSHIP.md (Rev 5: Controlled Open Source Governance)

Dokumen ini mendefinisikan kepemilikan dan hak kekayaan intelektual (IP) dari Mocheny Project. Dokumen ini adalah bukti digital (timestamped via Git) yang menyatakan hak kontrol penuh Pemilik.

---

### 1. Deklarasi Kepemilikan (Prior Use)

1.1. **Pemilik Aset:** Semua Intellectual Property (IP), Trademark, dan Copyright yang terkandung dalam Mocheny Project (Kode, Dokumentasi, Desain, dan Aset Visual) adalah milik eksklusif **Indra Wahyudi** (sebagai pendiri dan satu-satunya Pemilik IP) yang diwakilkan secara teknis oleh akun GitHub **itwahjoedi** (selanjutnya disebut sebagai **Pemilik**).

1.2. **Tanggal Pendirian:** Mocheny Project didirikan dan kepemilikan IP pertama kali dideklarasikan pada **11/11/2025**.

1.3. **Namespace Resmi:** Semua aset resmi Mocheny hanya didistribusikan melalui Organization Namespace yang dikontrol penuh oleh Pemilik:
    * **Kode:** `github.com/mocheny-project`
    * **Artifacts:** `ghcr.io/mocheny-project`
    * **Dokumentasi Sementara:** `mocheny-project.pages.dev`

### 2. Lingkup Kepemilikan Inti (Core Assets)

Pemilik memegang kendali penuh atas, namun tidak terbatas pada, aset-aset inti berikut:

* **Merek:** Nama "Mocheny", "Mocheny Project", dan semua kombinasi Branded Product (e.g., Mocheny CLI, **Mocheny Envabox, Mocheny Edna**).
* **Aset Visual:** Logo, Brand Identity, dan visual assets lainnya yang berada di repo `mocheny-project/mocheny-assets`.
* **Kode Inti:** Seluruh source code yang berada di bawah repositori **`mocheny-project/mocheny-cli`**, **`mocheny-project/mocheny-envabox`**, dan **`mocheny-project/mocheny-edna`**.
* **Nomenklatur:** Terminologi yang didefinisikan dalam **BRANDING.md** (Pipeline, State, Parity, Observer, Artifact, Inspect).

### 3. Kebijakan Lisensi (Controlled Relicensing)

3.1. **Lisensi Kode Saat Ini:** Seluruh source code (kecuali dinyatakan lain) dilisensikan di bawah **Lisensi Apache Versi 2.0**. Lisensi ini diberikan kepada pengguna, komunitas, dan kontributor asalkan mereka mematuhi ketentuan Trademark Mocheny Project.

3.2. **Lisensi Aset Non-Kode:** Dokumentasi dan Konten Naratif dilisensikan di bawah **Creative Commons Attribution 4.0 International License (CC BY 4.0)**. Namun, Brand Assets, Logo, dan Aset Visual inti memiliki **Copyright Pemilik** dan tidak tunduk pada lisensi terbuka apa pun, kecuali secara eksplisit diizinkan dalam BRANDING.md atau file LICENSE yang terpisah.

3.3. **Hak Relisensi Eksklusif:** Pemilik secara tegas menyatakan dan mempertahankan hak tunggal dan eksklusif untuk, sewaktu-waktu:
    * **Mengubah Lisensi** source code Mocheny Project (termasuk kode masa lalu, sekarang, dan masa depan) ke lisensi yang berbeda, termasuk model lisensi Open Core atau **Business Source License (BSL)**.
    * **Membuat Turunan Komersial** yang hanya tersedia di bawah Lisensi Komersial.

### 4. Kebijakan Kontribusi

4.1. **Perjanjian Kontributor:** Dengan mengajukan Pull Request (PR) ke Mocheny Project, kontributor secara otomatis menyetujui bahwa kontribusi kode mereka dilisensikan di bawah Lisensi Apache 2.0 dan **tidak mengubah atau menantang hak Pemilik** untuk Relisensi Eksklusif yang tercantum pada Bagian 3.3.

4.2. **Penggunaan Nama:** Kontributor tidak mendapatkan hak Trademark atau hak Branding apa pun atas nama "Mocheny" atau produk turunannya.

4.3. **Eksklusi Lisensi Governance:** File **OWNERSHIP.md** dan **BRANDING.md** secara eksplisit **dikecualikan** dari Lisensi CC BY 4.0 atau Lisensi Apache 2.0. Hak cipta file-file ini sepenuhnya dimiliki oleh Pemilik dan tidak dapat dimodifikasi atau didistribusikan ulang (kecuali untuk tujuan referensi) tanpa izin tertulis.
